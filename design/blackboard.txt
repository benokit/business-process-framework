# app config

app configurations specify static properties that determine a behavior of the application.
app configurations should be specified in elements.

kind: app-config/{specific}

Each module can declare its own configuration interface with schema id: app-config/{specific} 
and a default specific app-config element.

The runtime module, exposes a lazily instantiated object appConfig which is constructed from a collection all elements of kind app-config
merge in the same order as they were registered into a single object.

App config is accessible from _ctx.appConfig everywhere.

-----------

Implement the app config functionality.

In modules where currently an env variable is used, use app config instead.
