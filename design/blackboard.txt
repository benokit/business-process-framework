queries/create-entities-table.eson.sql

```sql
CREATE TABLE IF NOT EXISTS entities (
            id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
            entity_type TEXT NOT NULL,
            business_key TEXT NOT NULL,
            revision INTEGER NOT NULL DEFAULT 1,
            version INTEGER NOT NULL DEFAULT 1,
            data JSONB NOT NULL DEFAULT '{}',
            state JSONB NOT NULL DEFAULT '{}',
            timestamp_utc TIMESTAMPTZ NOT NULL DEFAULT NOW(),
            UNIQUE (entity_type, business_key)
        )
```

/infrastructure/database-modeling

data model service (kind: service)

method:
    - createModels (db-type):
        models = getElementsOfKind (data-model/{db-type})
        driver = getElement(db-driver-{db-type})
        for each model in models ordered by model.data.order ascending:
            service driver execute model.data.command
