# entity service extension

The idea is to replace entity component elements with a more explicit and direct elements.

Element kind: service/entity-service-extension/{entity-type}

The structure of the element is of the kind service.
Entity service extensions methods may implicitly assume to have entity record in the context _ctx.entity

The service can be called as any other service, but it can be called through the entity-service method execute, 
which expects the following input parameters:

- entityType
- businessKey
- revision
- method 
- methodInput

The service executes the first method which matches :method from a set of 
registered entity service extensions for the given entity type. 
Before the method invocation the service reads the entity and sets the _ctx. 
If a revision of the record and the If-Match do not match, the service throws an error.

The entity service extension methods are automatically exposed by the entity http API:

endpoint url /entities/:entityType/:businessKey/:method
