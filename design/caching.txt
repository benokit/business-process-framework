# Caching

## Execution node template:

{
    withCache: {
        cache: local-l1,
        cacheKey: { $join: [ user-profile/, #.user.id ]},
        validate: { $eq: [ #.cached.revision, #.user.revision ]},
        value: {
            method: get-user-profile
        }
    }
}

Note:
- cacheKey is lambdajson expression 
- validate is lambdajson expression applied on a cached value and input returning boolean: 
    - if false, a value is computed again and the cache value is replaced
    - if true, cached value is returned
    - if validate property is undefined, it defaults to _ => true
- value of the value property is an execution pipeline whose return value is cached and returned from the execution node


## cache element 

{
    kind: service/cache,
    id: local-l1,
    data: {
        interface: {
            get: string,
            set: {
                !key: string,
                value: any
            }
        },
        implementation: {
            get: { low: { module: ...}}
            set: { low: { module: ...}}
        }
    }
}
